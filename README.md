# Movie Sales Analysis Dashboard

## Project Overview
This project presents a Movie Sales Analysis Dashboard built using Power BI.  
It provides insights into movie performance, revenue trends, and sales distribution across different categories.

The goal of this project is to analyze movie sales data and help stakeholders make data-driven decisions.

---

## Features
- Revenue analysis by movie
- Genre-wise performance
- Regional sales insights
- Interactive dashboards
- Filters for dynamic analysis

---

## Tools & Technologies
- Power BI (.pbix)
- Data Visualization
- Data Analysis

---

## Project File
- Movie Sales Analysis.pbix (Main Power BI dashboard file)

---

## Key Insights
- Identifies top-performing movies based on revenue
- Shows trends in sales over time
- Highlights most popular genres
- Provides region-wise comparison

## Use Cases
- Business decision-making
- Sales performance tracking
- Entertainment industry insights
- Data analysis portfolio project

2. Open it using Microsoft Power BI Desktop  
3. Explore the dashboard using filters and visuals
